# Capstone kenzie q2

Essse projeto, trata-se do trabalho final a ser entregue no segundo modulo do curso de fullstack da Kenzie Academy.

## Tecnologias usadas

1. React
1.1.  Chakra UI
1.2.  React-router-dom
1.3.  ContextApi
1.4.  Yup - Hook - Form
1.5.  Axios
1.6. ChartJS
1.7. Lottie
